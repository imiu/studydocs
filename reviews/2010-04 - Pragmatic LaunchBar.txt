************
* Launching *
************
(Ctrl + Space) + Space      - show current application list
(Ctrl + Space) + Down       - show recent items list (Ctrl + N also works fine)


****************
* After launch *
****************
. or Cmd + L                - Enter url address
'                           - Enter text
;                           - Save text
Cmd + Alt + A               - Assign abbreviation
Ctrl + Space                - Start from the beginning


* Navigation *
Ctrl + H,J,K,L              - Like in Vim ^_^ (folders, text files, bookmarks (shows same domain bookmarks), emails (same domains))
Ctrl + Shift + H,J,K,L      - Home, Page Down, Page Up, End
Ctrl + Alt + L              - Show hidden files in ongoing directory and !!Application package content!!

* Application processing *
(Ctrl + Space) + Space      - show current application list
Cmd + R                     - Show current application list
Cmd + Q                     - Quit application
Cmd + H                     - Hide application
Cmd + Opt + F               - Force quit application

* Text manipulation *
Cmd + V                     - paste text from clipboard
Ctrl + Space (hold)         - instant send of selected text

* Actions *
Cmd + Y or Space            - open quick look
Cmd + D                     - "Drag" selected in Finder on LaunchBar target
Cmd + I                     - Get info
Cmd + T                     - Open in terminal
Cmd + Opt + T               - Run terminal command here
Cmd + Shift + T             - Run terminal command with selected item

Return                      - Open
Alt + Return                - Open & Hide others
Cmd + Return                - Show in Finder
Cmd + Shift + Return        - Show in current Finder window
Ctrl + Return               - Open all items inside a folder

Cmd + Shift + N             - Create new folder
Cmd + Shift + D             - Duplicate a file
Cmd + Delete                - Move to trash
Cmd + Shift + R             - Rename file/folder

Cmd + E                     - Eject disk

Cmd + Return                - Move to folder
Opt + Return                - Copy to folder
Cmd + Opt + Return          - Make and alias
Ctrl + Return               - Make symbolic link (absolute)
Ctrl + Opt + Return         - Make symbolic link (relative)
Ctrl + Shift + Return       - Make hard link

Cmd + C                     - Copy file's path
Cmd + Shift + C             - Copy & Paste text from LaunchBar to the foremost app
Cmd + Shift + V             - Paste files in LaunchBar to operate them

Shift + Space               - Repeat last search in other engine with this engine
Shift + Return              - Repeat last search with this engine

Ctrl + Return               - Open AppleScript file in editor
Opt + Return                - Run AppleScript in background

man, netstat, ping,
ssh, top, traceroute        - Can be done from LaunchBar and we can add more

Ctrl + Right Arrow          - Show available actions

post-http://srarch.ru/?s=*  - Perform a post request

Cmd + Ctrl + Return         - Open in GMaps contact's address

/ -> Cmd + Shift + C        - Open "Go to folder" menu from Open, Save dialogs,

* Calendar & Me *
Car service @ Feb 15	    - Creates an all-day event
Meeting @ Jun 2 2010	    - Creates an all-day event
Meeting @ Jun 12 3-5pm	    - Creates an event with a start and end time
TV Show @ next sun 20.15	- Creates a 1 hour event at the specified time
Trip to Paris @ Mar 13 5d	- Creates a multiday event that spans 5 days
Holiday @ Mar 13 2w	        - Creates a multiday event that spans 2 weeks
John @ tomorrow 2pm 15m!	- Creates an event with an alarm 15 minutes before
Dad's birthday @ Jun 12 !5d	- Creates an event with an alarm 5 days before


Buy milk	                - Creates a To Do
Buy butter !!	            - Creates a To Do with medium priorty
Clean the house > next sun	- Creates a To Do with a due date
Pay taxes > May 15 !3d	    - Creates a To Do with an alarm 3 days before due

tomorrow	                - Tomorrow, all-day

next fri	                - Next Friday

Aug 17 2011	                - August 17th, 2011
dec 24 8-10pm               - December 24, 8 PM - 10 PM
Fri 8:10 30min              - Next Friday, 8:10 AM - 8:40 AM
Oct 11 5d	                - 5 days, from October 11 to October 15

* Spotlight & Me *
Cmd + Shift + F             - Invoke Spotlight window search
rose	                    - rose
image rose	                - kind:image rose
todo	                    - kind:todo
"todo"	                    - todo
today todo	                - date:today kind:todo
todo next week	            - kind:todo date:next week
folder backup	            - kind:folder backup
url apple	                - kind:bookmark apple
pref tcp	                - kind:preferences tcp
ical tomorrow	            - kind:event date:tomorrow
mail meeting	            - kind:email meeting

* Clipboard & Me *
Cmd + \                     - Activate clipboard history
Cmd + Alt + \               - Show clipboard history
Cmd + Alt + Ctrl + V        - Paste and remove from clipboard history

Cmd + Shift + H             - Toggle hiding LaunchBar

*********
* Index *
*********
Cmd + Opt + I               - Show index window
Cmd + Shift + I             - Show item in index
