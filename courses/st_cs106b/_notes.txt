* C++ strings vs C-string *
string literals are actually old-style C strings
    from c++ to c
        s.c_str()
    from c to c++
        string("abc")
!! It bytes us when dealing with files. If we got a filename in a string, we need to convert it to C string
!! Concatenation don't work with pure two C strings, i.e.
"str1" + "str2" // won't even compile
"str1" + 'a'    // compiles, but doesn't work
"str1" + ch     // same stuff
